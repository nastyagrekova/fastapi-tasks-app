# Интелигентен помощник за планиране на задачи
## Стартиране:
pip install -r requirements.txt
python -m uvicorn app:app --reload

## Описание
Това е мое уеб приложение, създадено с цел да помага при организацията на ежедневни задачи.  
Системата позволява да се добавят и управляват задачи, да се задават приоритети, категории и срокове,  
а също така предлага интелигентни съвети и вдъхновяващи цитати.  

Към това добавих и интеграция с **OpenWeather API**, за да може приложението да показва актуална прогноза за времето.  
Така, при планиране на задачи, потребителят може да вземе предвид и условията навън — например дали е подходящо да се свършат външни дейности.

Проектът следва MVC архитектура:
- **controllers/** – съдържа логиката и маршрутите (FastAPI endpoints)
- **models/** – описва структурата на данните и базата
- **views/** – HTML шаблоните с Jinja2
- **static/** – CSS и статични ресурси

## Mock AI функционалности

В проекта са реализирани няколко симулирани интелигентни функции:

**AI съветник за задачи**

Анализира списъка със задачи и дава препоръка според броя и приоритета им.

Пример: „Започни с най-важните задачи – тези с приоритет 1.“

**Генератор на вдъхновение**

Извлича реални цитати чрез външно API (ZenQuotes).

Добавя и собствени мотивационни съвети (mock AI поведение).

## Външни API

**ZenQuotes API** – използва се за зареждане на вдъхновяващи цитати.

**OpenWeather API** – извлича текуща прогноза за времето по град.

## Използвани AI помощници
По време на разработката са използвани следните интелигентни инструменти:
- **ChatGPT** – за идеи, структура на кода и обяснения  ;
- **GitHub Copilot** – за подсказки при писане на функции и шаблони;
- **Mock AI функции** – за симулиране на „умно“ поведение без реален машинен интелект.
